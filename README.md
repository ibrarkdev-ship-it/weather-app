# 🌤️ Weather App

A simple Weather App built using **HTML, CSS, and JavaScript**.
It fetches real-time weather data using the **OpenWeatherMap API**.

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Display temperature in Celsius
- ☁️ Show weather description
- 🌤️ Display weather icon
- ⌨️ Search using the Enter key
- ⏳ Loading indicator
- ❌ Error handling for invalid city names

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- OpenWeatherMap API

---

## ⚙️ Setup

1. Get your free API key from **OpenWeatherMap**.
2. Open `script.js`.
3. Replace:

```javascript
YOUR_API_KEY
```

with your own API key.

4. Open `index.html` using **Live Server**.
